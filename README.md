# Plantarium

Plantarium is a React application designed to help users track and manage the growth of plants in their garden. The application allows users to register plants, monitor their growth, set care reminders, and access detailed plant care information.

# Features

- Plant Registration: Add new plants to your garden with details such as name, type, and planting date.
- Growth Tracking: Upload photos and notes to track the growth of each plant over time.
- Reminders: Set reminders for tasks such as watering, fertilizing, and pruning.
- Plant Care Information: Access a database of plant care tips and best practices.
- Activity History: Keep a log of all gardening activities for each plant.

# Tech Stack

Frontend: React
Backend: NestJS
Database: SQLite
Styling: CSS/SCSS
Deployment: Vercel/Netlify (Frontend), Heroku/AWS (Backend)
